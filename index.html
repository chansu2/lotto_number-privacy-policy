로또번호 생성기 개인정보 처리방침

시행일: 2025-11-05 · 마지막 업데이트: 2025-11-05 · 버전 1.0
운영자: 스왈로테일 · 연락처: hello@swltail.com
Google Play 공개용

본 처리방침은 로또번호 생성기(이하 “앱”)이 수집·이용·공유하는 개인정보와 이용자의 권리를 설명합니다. 앱은 필수 기능 제공에 필요한 최소한의 정보만 수집하며, 광고·분석·품질 개선을 위해 제3자 SDK를 사용할 수 있습니다.

1. 수집하는 정보

직접 제공 정보

  • 문의/피드백 시: 이메일 주소, 문의 내용 등

자동 수집 정보

  • 기기 정보: 모델명, OS 버전, 앱 버전, 언어/국가, 광고 ID(AAID)
  • 이용 로그: 화면 조회, 버튼 클릭, 세션 길이, 충돌 로그
  • 알림 관련: 예약된 로컬 알림 ID, 시간대(timezone)

2. 수집 방법

  • 앱 사용 중 자동 수집(분석/Crashlytics/광고 SDK)
  • 이용자 입력(문의, 공유 기능 사용 등)

3. 이용 목적

  • 핵심 기능 제공 및 설정 저장(hive, shared_preferences)
  • 알림 제공(flutter_local_notifications, timezone)
  • 앱 성능 및 사용성 분석(firebase_analytics)
  • 오류 진단 및 안정성 향상(firebase_crashlytics)
  • 광고 제공 및 광고 성과 측정(google_mobile_ads)

4. 사용 중인 SDK 및 제3자 제공

아래 SDK는 개인정보를 수집·처리할 수 있으며 각 공급자의 정책이 적용됩니다.

  • Google Mobile Ads SDK: 맞춤형/비맞춤형 광고 제공, 광고 ID 사용. 광고 정책
  • Firebase Analytics: 이벤트, 디바이스 정보 수집. Firebase 개인정보 보호
  • Firebase Crashlytics: 충돌 보고서, 디바이스 상태, 스택 트레이스. Crashlytics 데이터

법령이 허용하는 범위 내에서 서비스 제공자, 클라우드·로그 분석 사업자에게 처리 위탁이 이뤄질 수 있습니다. 위탁 시 안전성 확보 조치를 요구합니다.

5. 로컬 저장소 사용

  • Hive / Hive Flutter: 앱 내 설정, 캐시 데이터. 민감정보 저장 없음.
  • Shared Preferences: 간단한 키-값 설정. 비밀번호 등 민감정보 저장 금지.
  • flutter_local_notifications + timezone: 로컬 알림 스케줄과 시간대 정보.

6. 광고 및 맞춤형 선택

  • 맞춤형 광고 동의가 필요한 지역(EU 등)에서는 최초 실행 시 동의를 요청합니다.
  • 비맞춤형 광고를 선택할 수 있습니다. 기기 설정에서 광고 ID 재설정/맞춤광고 제한 가능.

7. 보관 기간

  • 로컬 설정: 이용자가 앱을 삭제하거나 초기화할 때까지
  • 분석/충돌 로그: 목적 달성 후 지체 없이 파기 또는 익명/집계 처리 (Firebase 보존 정책에 따름)

8. 아동의 개인정보

본 앱은 원칙적으로 만 14세 미만(또는 현지법상 아동)에 대한 직접적 서비스 제공을 목표로 하지 않습니다. 아동 이용이 예상되는 경우, COPPA/AGE 관련 요구사항에 부합하도록 비맞춤형 광고만 제공하거나 광고를 제한합니다.

9. 이용자의 권리

  • 열람·정정·삭제·처리정지 요청: [연락처 이메일]
  • 광고/분석 동의철회: 앱 내 설정 또는 최초 동의 화면, 기기 설정에서 관리
  • 데이터 이동성·반대권: 관련 법령이 정한 범위 내에서 행사 가능

10. 국외 이전

Firebase, Google Ads 등 클라우드 서비스는 해외 서버를 사용할 수 있습니다. 이 경우 관련 법령이 요구하는 보호 조치를 이행합니다.

11. 보안 조치

  • 전송 구간 암호화(HTTPS)
  • 접근 통제, 최소 권한 원칙
  • 민감정보 로컬 저장 금지

12. 기기 권한

현 버전 기준 필수 권한 없음. 선택 권한은 기능 사용 시점에 동의 요청:

  • 알림 권한: 로컬 알림 제공
  • 공유 기능 사용 시, 시스템 공유 시트 접근

13. 타사 링크

앱에는 외부 사이트로 연결될 수 있으며, 해당 사이트의 정책이 적용됩니다.

14. 정책 변경

법령·서비스 변경에 따라 본 방침이 수정될 수 있습니다. 중요한 변경은 앱 내 공지 또는 본 문서 상단의 “마지막 업데이트” 갱신으로 알립니다.
